我是champ，正是计算机大二学生。这是一个利用qt写的植物大战僵尸游戏（带有个人改编）。模仿借鉴于大佬geekexplorer的PVZ项目。

Hi, my name is champ,currently a cs student. this is a plant vs zombie game developed by c++ and qt(with personal reference).

Inspired by master geeeekexploer


功能：原版具有的所有功能，加上自创两个按键


按键1：freeze，一键减速场上所有僵尸


按键2：power，增强植物/僵尸的能力（高度扩展性，可高度自定义）

对土豆地雷使用“power”：瞬间进入准备状态，并且恢复卡片冷却时间

Use 'power' on potatoes to instantly enter preparation mode and restore card cooldown time



对寒冰射手使用“power“：发射出五颗能完全冻住僵尸的豌豆

Use 'power' on the Ice Shooter: Launch five peas that can completely freeze zombies


对坚果使用“power”：在一定时间段内恢复血量，最多恢复至原血量的两倍

Use "power" on nuts: restore blood volume within a certain period of time, up to twice the original blood volume


对僵尸使用“power”：增加僵尸的速度

Use "power" on zombies: increase zombies' speed


其他植物和僵尸的个性化power还有待实现

目标是加上更多的植物和僵尸种类，如：


1.圣坚果：在被攻击时发出圣光，小幅度恢复hp，对吸血鬼僵尸有奇效。
